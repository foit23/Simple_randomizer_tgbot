#SIMPLE RANDOMIZER DICE TELEGRAM BOT

##This is a simple Telegram bot that, when used in your groups, allows you to use the /Reroll command to roll a dice, which will land on a random side. Nothing fancy, just simplicity and convenience.

##How to download?
1. download python, version recommended 3.13+
2. add python to path (optional, recommended)
3. enter into terminal:

  3.1 pip install aiogram

  3.2 pip install asyncio

##Add the bot to the group and enter /Reroll in the chat
